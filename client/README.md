# Eshaz Dream World Client

React 19 client application built with Vite, Tailwind CSS, React Router, Redux Toolkit, Axios, Framer Motion, React Icons, and React Hot Toast.

## Commands

- `npm.cmd run dev` starts the local Vite server.
- `npm.cmd run build` creates an optimized production build.
- `npm.cmd run lint` checks the source with Oxlint.
- `npm.cmd run preview` previews the production build locally.

Configure the API base URL with `VITE_API_URL` in `.env`.
